# cosmoFits 🪐

## El proyecto cosmoFits permite la lectura y modificación de los archivos .fts

### Language:
- 🇬🇧 [English](./README.md)
- 🇲🇽 Español

## Instalación

### Clone the project:

```
git clone https://github.com/codingTeach/cosmoFit
```

Dirigete al directorio del proyecto e instala las dependencias:

```
cd cosmoFit && npm i
```

### Corre cosmoFits 🔥

```
npm run dev
```
